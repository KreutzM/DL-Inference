# MVP JAWS Support Knowledge Base

This is the only seeded knowledge base for the first runnable RAG slice.

## Reset settings

If a user asks how to reset JAWS settings, the support path is:

1. Close JAWS.
2. Rename or back up the current settings folder.
3. Restart JAWS so it creates a fresh settings profile.
4. Reapply only the customizations that are still needed.

## Troubleshooting notes

- Prefer the simplest reset that solves the issue.
- If the problem is caused by a recent profile change, mention that the user can test with a clean profile.
- If evidence is missing, say that the repository does not yet document a better answer.
